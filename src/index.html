Minhas obeservações

DOCTYPE html

Antes de iniciar o documento html precisamos passar a sua configuração de versão. A tag acima diz para o browser qual versão do html está escrito

Tag html

É a tag html que inicia o documento e também aonde passamos algumas configurações como linguagem.

Tag Head

Aqui encontramos as tags meta que são utilizadas para passar determinadas configurações que o browser utiliza e fica invisíveis para os usuários.

Também utilizamos a tag title que é utilizada para dar títulos as páginas.

Dentro dessa tag também podemos importar css através das tags link.

Tag Body

É dentro dessa tag que fica todo o conteúdo do nosso html visualizado pelo usuário.

Tags inline e block

Inline

São as tags que utilizam a sua própria largura e não causam uma quebra
de linha

Block

São as tags que um comportamento de caixa e possuem a quebra de linha pois a sua largura ocupa a tela toda.

Tags div

Possuem o comportamento de bloco, não possui significado e são utilizadas para auxiliar na estilização do css

Tags span

Possuem o comportamento inline, não possui significado e são utilizadas para auxiliar na estilização do css

Tags h1 à h6

São tags que servem para descrever uma seção indo com hierarquia do maior para o menor

Tag p

São tags que servem para descrever um parágrafo e tem como estilo padrão block

Tags em & i

São tags utilizadas para transformar um texto em itálico sendo que em acresenta ênfase ao texto

Tag a

Utiliza para colocar links nas páginas e também para servir como âncoras para elementos no mesmo documento HTML

Tag img

Utilizado para carregar imagens no HTML. O atributo alt serve como texto alternativo para caso a imagem não carregue e também ajuda na acessibilidade. Quando o leitor de tela dá foco a imagem o texto do alt é falado.
